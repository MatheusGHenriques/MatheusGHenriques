## Projects

### [Crisma PSVP](https://github.com/MatheusGHenriques/Crisma-PSVP)
Offline-first peer-to-peer communication platform built with Flutter/Dart for environments without internet access.
- UDP peer discovery, TCP transport, automatic reconnection
- End-to-end encryption - ECDH X25519 + AES-GCM + Argon2id key derivation
- Group-based authentication with secure key persistence (Secure Storage)
- Encrypted local storage, real-time messaging, task management, polls and PDF sharing

### [HIPS SSHD Guard](https://github.com/MatheusGHenriques/HIPS-SSHD-Guard)
Lightweight Host Intrusion Prevention System written in Bash for SSH hardening for Linux servers.
- Real-time log monitoring via `journalctl` or `auth.log`
- Brute-force detection with configurable time window and failure threshold
- Automatic IPv4/IPv6 blocking and unblocking via `iptables`/`ip6tables`
- Persistent event logging and auto-generated configuration

### [JavaChess](https://github.com/MatheusGHenriques/JavaChess)
Desktop chess application built in Java with Swing/AWT.
- Move validation, check/checkmate detection, pawn promotion
- Object-oriented architecture with encapsulated game state
- Swing/AWT desktop GUI with dynamic resizing and local persistence
